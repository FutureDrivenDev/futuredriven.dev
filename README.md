# futuredriven.dev

Landing pages and blog engine for Future Driven sites:

- futuredriven.dev
- futuredriven.blog
- futuredriven.capital

Runs on Deno and Deno Deploy.

### Running locally

Create a `.env` file to add tokens and enable live reload:

```
BUILDER_API_KEY=...
NOTION_TOKEN=...
NOTION_POSTS_DB=...
ENABLE_LIVE_RELOAD=true
```

Run:

`deno task dev`
